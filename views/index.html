FreeCodeCamp API Project: URL Shortener Microservice

User stories:

  • Passing a URL as a parameter will retrieve a shortened URL in the JSON response.
  • If an invalid URL is passed that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When the shortened URL is used, it will redirect the user to the original link.

Example creation usage:

  • https://md-url-shortener-microservice.glitch.me/new/https://www.google.com
  • https://md-url-shortener-microservice.glitch.me/new/http://www.bing.com

Example creation output

{ "original_url":"www.bing.com", "short_url":"https://md-url-shortener-microservice.glitch.me/8170" }

Usage:

https://md-url-shortener-microservice.glitch.me/8170

Will redirect to:

https://www.bing.com/
